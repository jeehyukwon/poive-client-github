# Poive

To start, install npm modules :

```bash
$ npm install
```

Run development environment :

```bash
$ npm run dev
# Run Electron App with ./build/server.js
$ npm run app-dev
```

Build :

```bash
$ npm run build
# Run Electron App with ./app/dist/index.html
$ npm run app-prod
```
