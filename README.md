# Poive

시작하기 위해 다음과 같이 Node 모듈을 설치합니다 :

```bash
$ npm install
```

다음과 같이 개발환경을 실행합니다 :

```bash
$ npm run dev
# Electron App을 실행합니다 (./build/server.js로 실행)
$ npm run app-dev
```

다음과 같이 Build합니다 :

```bash
$ npm run build
# Electron App을 실행합니다 (./app/dist로 실행)
$ npm run app-prod
```

---

Poive는 [VuePack](https://github.com/egoist/vuepack) 으로 만들어진 프로젝트입니다.
